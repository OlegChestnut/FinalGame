# FinalGame
justfinalproject
Техническое задание

Требуется разработать приложение windows forms, для сдачи.
// ?\_(?)_/?
Приложение должно быть реализовано на языке программирования C#. В приложении должна быть реализована возможность хранения данных о победе/проигрыше игрока. 

В приложении должны быть реализованы несколько окон.

Для хранения данных должны быть использованы текстовые файлы формата txt.

Разработанное приложение должно демонстрировать пользователю предварительный игровой процесс.

Форма включает в себя реализацию переходов между формами\экранами; передачи данных между формами\экранами;

На формах должны быть реализованы элементы управления Button, Label, TextBox, CheckBox, PictureBox.

Все экранные формы приложения должны соответствовать единому стилю.
